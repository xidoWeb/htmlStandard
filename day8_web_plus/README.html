사용법

  1. scss폴더에 *.scss 파일 작성하면, 자동으로 같은 레벨(scss와 동일레벨)의 폴더 css폴더에 저장해준다.
  2. css와 동일하게 사용 가능!
  3. 변수를 사용해서 다양한 지정된 내용을 기록해서 사용할 수 있다.( $변수명)
  4. 자식선택자, 자손선택자 사용시!! 여러번 반복사용기법을 줄일 수 있다!
  5. @mixin이라는 기능을 이용해서 반복하는 내용을 축약으로 불러올 수 있다.
